# GrowTrack -- Productivity Workspace

GrowTrack is a web-based productivity workspace designed to help users
track habits, visualize progress, and manage daily productivity.\
The project is built using HTML, CSS, and JavaScript with LocalStorage
to persist user data without a backend.

------------------------------------------------------------------------

## Features

-   Add and track daily habits
-   Mark habits as completed
-   Habit analytics with charts
-   Calendar productivity view
-   User profile management
-   Dark / Light theme support
-   Dashboard overview for quick statistics

------------------------------------------------------------------------

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript
-   Chart.js
-   LocalStorage

1.  Download or clone the repository
2.  Open the project folder
3.  Run **index.html** in your web browser

Clone command:
